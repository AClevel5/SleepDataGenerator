Problem: Apple health stopped tracking my sleep via a sleep schedule and I very much dislike wearing a watch to bed.
Solution: This.

Use this link to download a premade shortcut to import sleep data: https://routinehub.co/shortcut/13360/

To Use:
1. Download project and review variables at top and modify them to the dates you are trying to achieve.
2. Run script.
3. Copy console into CSV.
4. Create a folder on Icloud and save your CSV to it.
5. Go to the link above to download the shortcut.
6. Hit run on the short cut and select your CSV file.
